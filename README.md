# 🌦️ Weather App
A clean and responsive weather application built with **HTML, CSS, and JavaScript**.  
This app allows users to search for any city and view real-time weather information instantly.

## 🔗 Live Demo

🚀 https://wahidahmadrus.github.io/Weather-APP/

---

## ✨ Features

- 🌍 Search weather by city name
- 🌡️ Display real-time temperature and weather conditions
- ⚡ Fast and responsive user interface
- 📱 Mobile-friendly responsive design
- 🔄 Dynamic weather updates using JavaScript
- 🎨 Simple and clean UI

---

## 🛠️ Technologies Used

- **HTML5** — Structure of the application
- **CSS3** — Styling and responsive layout
- **JavaScript (Vanilla JS)** — App logic and API integration

---

## 🚀 How It Works

1. Enter the name of a city
2. Click the search button or press **Enter**
3. The app fetches real-time weather data from an API
4. Weather details are displayed instantly on the screen

2️⃣ Navigate to the project folder
```bash
cd Weather-APP
```
3️⃣ Open the project

Open index.html in your browser.
